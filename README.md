## Integration-Gemini-AI-API
Integration-Gemini-AI-API adalah program backend yang dirancang untuk mengintegrasikan teknologi Gemini AI dalam pengolahan berbagai jenis data, seperti teks, gambar, audio, dan dokumen. Aplikasi ini memungkinkan pengembang untuk membangun layanan berbasis kecerdasan buatan yang mampu melakukan analisis, generasi, dan pemrosesan konten secara otomatis melalui satu sistem terpusat.

## Cara Penggunaan
1. **Clone Project ini**
```bash
git clone https://github.com/LRMNTRIX03/Integration-Gemini-AI-API/
```
2. **Setup Env anda dan sesuaikan key gemini api key dari akun anda**
```php
PORT=3000
NODE_ENV=development
GEMINI_API_KEY=Your Key
```
3. **Jalankan Server pada PORT 3000**
```bash
node index.js
```
## Dokumentasi API
1. **Teks**
   ```php
